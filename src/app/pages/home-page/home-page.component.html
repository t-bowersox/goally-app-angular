Head's up! This is just a demo app!

Goally is part of a portfolio project with the goal (pun absolutely intended) of demonstrating my ability to create full-stack apps using different frontend frameworks.

This app is fully functional and designed with security in mind, though any data you provide will be deleted after 24 hours of inactivity. I both don't want to store any info about you longer than needed, and also want to keep this project affordable for me to host.

About this version

This version of Goally was created using the Angular framework, along with the Angular Material component library. Although I used Sass as the stylsheet format, I made use of modern CSS (including custom properties) where appropriate.

This app includes a light and dark theme based on the user's OS preference. It also disables animations for users who prefer reduced motion.
